Easily integrate Indexic's aReservation Tour Booking and Rental Reservation Software into your WordPress website.  You can add booking buttons with simple stylable quick codes that work in most themes.Easily add FareHarbor reservation calendars, booking embeds, and buttons to your site.

This plugin will by default include javascript code (considered 3rd party code) from indexic's servers.

== Description ==

Indexic aReservation is cloud based software makes it possible to keep all the moving parts of your tour or rental business organized in one convenient place and running efficiently. From taking bookings, to signing waivers, scheduling your staff, tracking equipment availability, communicating with your clientele, and more, weâ€™re here to help your business scale without sacrificing the customer experience.

This plugin makes it easy to integrate the online booking process into you WordPress website.  A 3rd party script from indexic's aReservation website will be inserted into your WordPress website pages.  This script will allow for a seamless integration of the booking purchasing process on aReservation with your website.  If you prefer not to include the script, the quick codes will still work but will simply be links to the 3rd party website aReservation.com.

Since this plugin is designed to ease the integration of indexic's aReservation online booking process you must have an aReservation company account set up first for this plugin to be of any value to you.  If you don't already have your company set up you can visit https://indexic.net to learn about indexic's aReservation service.

== Installation ==

= From the WordPress plugin directory (recommended): =

1. Navigate to Plugins > Add New in your WordPress Dashboard
2. Search for "indexic"
3. Click Install on the "indexic aReservation" plugin
4. Activate the plugin

= By direct upload: =

1. Download the plugin and unzip it.
2. Upload the indexic-aReservation folder to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress

= Then =

1. Navigate to Settings > aReservation 
2. (optional - not recommended) If you would prefer not to include indexic's 3rd party javascript code, delete the script at the top of the aReservation settings page.  Without the script visitor will be redirected to the aReservation website when they click on a button provided by the short codes. The script allows visitors to remain on your website while going through the aReservation booking process.
3. Enter the URL friendly company name of your company provided by indexic.
4. Optionally style the button(s) you would like to use in your website
5. Each button type has a short code such as [ares_button id="standard"], place this short code on any pages you would like to have a booking button.
